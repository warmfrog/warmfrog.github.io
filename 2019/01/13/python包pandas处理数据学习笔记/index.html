import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

创建数据对象

s = pd.Series([1,3,5,np.nan,6,8])
dates = pd.date_range('20130101', periods = 6)
df = pd.DataFrame(np.random.randn(6,4), index=dates, columns = list('ABCD'))

df.loc[dates[0]]

  • 获取标量值

    df.loc[date[0],’A’]

  • 更快的获取标量：

    df.at[dates[0],’A’]

根据位置选择

  • 通过整数位置选择：

    df.iloc[3]

  • 切片：

    df.iloc[3:5, 0:2]
    df.iloc[[1,2,4],[0,2]]

    df.iloc[1,1]

  • 更快的获取标量:

    df.iat[1,1]
